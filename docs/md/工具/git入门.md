# git入门

## 初始化：

  ```
  git init
  ```
## 查看状态

  ```
  git status
  git diff
  ```
## 版本管理
* 基本: 工作区、暂存区、当前分支

* 查看
  ```
  git add <filename>
  git commit -m "<message>"   #提交快照,本地存档
  git log   #提交记录
  git reflog  #记录每一次指令
  ```
  
* 版本回退:
  
  ```
  #  head 表示当前版本 head^、head^^ 表示上和上上一版本 head~10 十个版本前
  git reset id
  git reset head
  git revert head
  ```
